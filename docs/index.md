# Welcome to Spectra

**Spectra** is an advanced security audit platform designed to streamline **Static Application Security Testing (SAST)** and **Dynamic Application Security Testing (DAST)**.

![Spectra Dashboard](https://placehold.co/800x400?text=Spectra+Dashboard)

## Key Features

*   **Unified Dashboard**: View code vulnerabilities (SAST) and web vulnerabilities (DAST) in a single pane of glass.
*   **Multi-Scanner Engine**:
    *   **SAST**: Integrates *Semgrep*, *Trivy*, and *Gitleaks* for comprehensive code analysis.
    *   **DAST**: Powers dynamic web analysis using *Nuclei* engine.
*   **Automated Scheduling**: Schedule daily or weekly scans for your repositories.
*   **Professional Reporting**: Generate and export audit reports in Word format with one click.
*   **AI-Enhanced**: Uses LLMs to generate Executive Summaries and analyze complex findings.
